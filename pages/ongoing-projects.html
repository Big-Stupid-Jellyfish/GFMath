Ongoing Projects

By BigStupidJellyfish | 2022-June-23 | Updated 2022-July-08

Science can be a slow process. There are a number of systems in Girls' Frontline I'm actively tracking, in the hopes of better understanding their rates and inner workings.

As these are a bit scattered throughout my articles, this page will serve as a dedicated list of all such projects. I welcome any and all data from other players, too, to help speed things up and get more accurate estimates. If you wish to contribute, don't hesitate to reach out: /u/BigStupidJellyfish_ or BigStupidJellyfish#6974 on discord. If you don't have an account for either of those, my feedback form is also open.

Core rates#

Page link.

After seeing various incorrect numbers thrown about for the core rates on different farming maps (and wasting ~2k runs on 11-5 because of an alleged 0.55 core rate), I've been compiling core and limited drop rates. If you've farmed any particular map a lot and use GFAlarm, please do send me your Username_UID_rescued_doll.csv file.

I'm also tracking core rates from autobattles. If you're running some for whatever reason and would like to contribute, the preferred submission format would be:

Map Attempts 5* 4* 3* 2* (optional)
7-1 3 0 0 1 9
7-1 3 0 0 2 6
7-1 3 0 0 2 10
11-5 3 0 0 4 14

where "attempts" is the number of clears executed at once. 3 max without an autobattle card.

Protocol Assimilation data#

Page link.

I'm tracking info on coalition unit sizes, shininess, and decoding practice keycode rates. With more data, I'd like to more precisely nail down XL/XS/shiny rates, and get keycode rates to sub-1% accuracy.

Preferred submission format example (unit info):

Unit name Size Shiny Rarity
Striker S 0 1
Manticore M 0 2
Vespid XL 1 1

Shiny is 0 (not shiny) or 1 (shiny). Rarity is the base rarity - 1 or 2. Ringleaders are excluded.

Preferred submission format example (keycodes): 0,1,0,1,0,0,0,0 (or any similar string/table/spreadsheet). 0 = basic keycodes, 1 = advanced keycodes.

Fairy talent rates - production & calibration#

Page link.

Not all talents are equally likely, both in calibration and production. I'd like to figure out exactly how common each one is, and what categories exist.

Preferred submission format example:

Fairy name (production) Resulting talent Fairy name (calibration) Resulting talent
Sniper Accuracy I Parachute Keen
Fury Critical I Parachute Keen
Command Armor II

If you are using an updated version of GFAlarm, it now saves fairy talents from production (but not calibration). A copy of its Name_UID_production.csv file would also work. Keep in mind that, if you get back-to-back talents of the same name in calibration, they should be recorded as separate entries.

2.09 off-target anchored production rates#

Page link.

I'm investigating whether or not the 4/5* drop rates are significantly different here than in normal crafting. If you'd like to contribute your data, please do check out this form. If you already did your crafting and forgot your results, but have GFAlarm saving production data, results are saved in the production csv under WISH-NORMAL.

Skin gacha rates#

Official numbers state a 2% overall skin rate. The distribution between skins, however, is not known. I've started a form here to collect gacha pulls, to see what's really going on.

SF dragging#

Page link.

I'm collecting data on the effectiveness of different SF HOC support combinations on 8-2. Assuming friend supports in SC get fixed eventually, that + Alina will make this mostly obsolete. The data recording process is a bit troublesome, so I'm mostly just mentioning this here for completeness.

Comments#

These pages will be periodically updated (1-2 times a month) as my data collection grows.

Data submissions must be newly acquired - for example, going back through one's existing fairy armory and reporting the talents will give a biased sample. Fairies with useful talents are more likely to be kept, and fairies with matching ones may have been fed into paras.
