    Core Rates

    By BigStupidJellyfish | Last updated 2021-November-17

    Cores are nice to have, so maps that give lots of cores must be nice. It takes something like 2000-3000 cores to craft a 5★ Para, so there's always a use for them - PR could happen this decade, and maybe even Singularity+ if we're lucky.

    General rates#

    Here's a list of popular core-farming maps and their estimated core rates.

    Map Core Rate Source
    4-3E 0.2 Limited personal data (7 cores/34 runs)
    0-2 0.4 Common claim
    0.40 Personal data (267 cores/672 runs)
    10-4E 0.88 Personal data (826 cores/936 runs)
    0.83 Link
    11-5 0.55 Common claim, likely incorrect
    0.38 Personal data (554 cores/1469 runs)
    11-6 1.24 Personal data (359 cores/290 runs)
    12-4E 0.90 Personal data (8484 cores/9465 runs)
    0.85 Link
    0.88±0.08 Link
    13-4 0.86±0.03 Link

    8-1N obviously gives no cores, but may be nice if you only want XP and want it fast. Heliport-spawned enemies generally don't drop dolls so 8-2 (SF dragging) doesn't give anything significant.

    Cores per run isn't the only metric you should use. XP is also valuable, and each run will have different costs/requirements/time per run.
